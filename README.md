# mdxpad

> the most powerful, performant, flexible, extendible, beautiful, modern, useful, user-friendly, robust, efficient, configurable, awesome markup (ie powered by `MDX`) editor around!

reqs:
- supports .md (eg GFM etc) & .mdx
- live preview
- extensible via plugins
- remark + rehype plugin (built-in + user installable) support
- MacOS app
